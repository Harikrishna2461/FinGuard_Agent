🤖 ML Pipeline

Hybrid 3-tier scoring engine:
① 13 Deterministic Rules (AML, sanctions, velocity, concentration) — 40% weight
② GradientBoosting Classifier (97% accuracy) + IsolationForest (93% accuracy) — 60% weight
③ LLM Fallback via Groq (llama-3.1-70b) for borderline scores (40-60)

🚀 System Architecture

Backend: Flask + SQLAlchemy  |  Agents: CrewAI (Sequential)  |  LLM: Groq API  |  Vector DB: ChromaDB  |  ML: scikit-learn
